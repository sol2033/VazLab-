### Отслеживание прогресса:
- Задачи хранятся в SQL базе (todos таблица)
- Можно запросить текущие готовые задачи через SQL

### Структура MVP:

```
Фаза 0: Настройка окружения (1-3 дня) ← ТЕКУЩАЯ ФАЗА
  ↓
Фаза 1: Bluetooth подключение (3-5 дней)
  ↓
Фаза 2: ELM327 протокол (4-6 дней)
  ↓
Фаза 3: База DTC кодов (2-3 дня)
  ↓
Фаза 4: UI диагностики (3-4 дня)
  ↓
Фаза 5: Тестирование (2-3 дня)
  ↓
Фаза 6: Сборка и релиз (1-2 дня)
```
