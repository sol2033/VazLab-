## Параллельная работа

Можно делать параллельно:
- **Фаза 3 (База DTC)** ➕ **Фаза 1-2** - база кодов не требует Bluetooth
- **Дизайн UI** можно начинать во время разработки Фазы 2

## Критические точки проверки

### ✅ Checkpoint 1 (День 3):
- Flutter запускается
- Проект создан
- Все зависимости установлены

### ✅ Checkpoint 2 (День 8):
- Bluetooth работает
- Видны ELM327 устройства
- Можно подключиться

### ✅ Checkpoint 3 (День 14):
- ELM327 инициализируется
- Читаются DTC коды
- Парсинг работает правильно

### ✅ Checkpoint 4 (День 18):
- UI показывает ошибки
- Можно сбросить ошибки
- Навигация работает

### ✅ Checkpoint 5 (День 21):
- Протестировано на реальном авто
- Основные баги исправлены
- Стабильно работает

### ✅ Checkpoint 6 (День 23):
- APK собран
- Установлен и работает
- Готов к распространению

## Риски и митигация

| Риск | Вероятность | Митигация |
|------|-------------|-----------|
| ELM327 клон с багами | Высокая | Купить качественный адаптер v1.5+ |
| Проблемы с Bluetooth разрешениями Android 12+ | Средняя | Изучить документацию заранее |
| ЭБУ не отвечает на команды | Средняя | Протестировать с Torque сначала |
| Нехватка описаний DTC для ВАЗ | Низкая | Собрать из chiptuner.ru |
| Flutter сборка не работает | Низкая | Следовать официальной документации |

## Метрики успеха MVP

### Технические метрики:
- **Время подключения**: < 10 секунд
- **Успешность подключения**: > 95%
- **Точность парсинга DTC**: 100%
- **Стабильность**: работа без сбоев 30+ минут
- **Размер APK**: < 15 МБ

### Пользовательские метрики:
- **Время до первой диагностики**: < 2 минуты (включая подключение)
- **Количество кликов до сканирования**: ≤ 3
- **Понятность без инструкции**: 90% пользователей
- **Время отклика UI**: < 100ms

### Функциональные требования:
- ✅ Подключение к ELM327 Bluetooth
- ✅ Чтение активных DTC
- ✅ Сброс DTC
- ✅ Описания на русском
- ✅ Темная тема
- ✅ Стабильная работа
